# CSS Basic Project

## Description

This project focuses on applying basic CSS styling and layout techniques to HTML pages. It demonstrates the use of external CSS files, Flexbox for page structure, responsive web design principles, and custom styling to improve the appearance and usability of webpages.

## Project Structure

```text
css_basic/
│
├── index.html
├── tweets.html
├── base.css
├── styles.css
└── README.md
```

## Features Implemented

### 1. CSS Integration

* Created and linked:

  * `base.css`
  * `styles.css`
* Applied both stylesheets to:

  * `index.html`
  * `tweets.html`

### 2. Flexbox Layout

Implemented a responsive page layout using CSS Flexbox:

* `<body>` displays content in a column layout.
* `<main>` displays content in a row layout.
* `<article>` occupies 2/3 of the available width.
* `<aside>` occupies 1/3 of the available width.
* Vertical scrolling enabled within `<article>` and `<aside>`.

### 3. Responsive Design

* Added the class `works_on_smartphone` to the `<body>` tag.
* Included the viewport meta tag for proper mobile rendering.
* Layout adapts smoothly to smaller screens.

### 4. Custom Styling

* Added personalized styling using `styles.css`.
* Enhanced visual appearance with custom colors, spacing, borders, and typography.
* Applied styling consistently across both webpages.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Responsive Web Design

Through this project, I learned how to:
